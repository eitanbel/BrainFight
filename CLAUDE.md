# BrainFight

## Nom du projet
BrainFight

## Ce que fait l'app
Quiz multijoueur en temps réel entre amis.

## Stack
- React
- Vite
- Firebase Realtime Database
- API Anthropic

## Commandes utiles
```bash
npm install       # Installer les dépendances
npm run dev       # Lancer le serveur de développement
npm run build     # Build de production
```
